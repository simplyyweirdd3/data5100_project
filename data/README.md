# Data folder contents

Files included:
- cardio_train.csv
- diabetes_prediction_dataset.csv
- healthcare-dataset-stroke-data.csv

Notes:
- All data files are the versions used in the project and are stored here.
- If a file is missing due to hosting restrictions, download from the linked sources below and place the CSV in this folder.
Kaggle sources:
- Diabetes: https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset
- Cardiovascular: https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset
- Stroke: https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

To reproduce results:
1. Ensure the CSV files are present in this folder.
2. Set DATA_PATH = "./data/" (or "../data/" if running from Notebooks folder) in the notebook top cell.
